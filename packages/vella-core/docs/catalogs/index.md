# Catalogs

Reference catalogs generated directly from live `vella.core` code by `scripts/generate_catalogs.py` and gated in CI (`--check`). They cannot silently drift from the package.

- [Edge vocabulary](edge_types.md) — canonical `EdgeTypes` constants.
- [Public API](public_api.md) — every `vella.core.__all__` symbol.
- [Compatibility policies](compat_policies.md) — the `CompatPolicy` matrix.
- [Type registry](type_registry.md) — types registered in `default_registry`.
